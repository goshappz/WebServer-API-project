# Сервис для просмотра и коллекционирования случайных фотографий собак со всего мира
Сайт подгружает случайную фотографию собаки через Dog CEO API. Зарегистрированный пользователь может сохранять фотографии к себе в профиль. 
Участники: Попов Георгий (Тимлид), Ушаков Иван
